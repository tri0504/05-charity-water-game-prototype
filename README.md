# Charity: water Game Prototype

This project is a browser-based memory matching game themed around clean water solutions. Players flip cards to find matching pairs, earn points for each match, and race against a running timer. Every successful match reveals a short educational fact about how safe water projects support communities.

## What the Application Does

- Displays a 4x4 memory board (8 matching pairs).
- Starts the timer on the first card click.
- Awards points for each successful match.
- Reveals a learning snippet tied to each matched solution.
- Shows a celebration overlay with final score and final time when all pairs are matched.
- Lets players reset and play again with a newly shuffled board.

## Technologies Used

- HTML5: page structure, game board container, score/timer display, and win overlay.
- CSS3: custom charity: water-inspired styling, card flip animations, responsive layout, and confetti effects.
- JavaScript (ES6): game state management, card matching logic, timer behavior, score updates, and DOM interactions.
- Bootstrap 5 (CDN): utility classes and basic responsive spacing/layout helpers.

## Assets

- Local images in the `img/` folder for branding and card visuals.
